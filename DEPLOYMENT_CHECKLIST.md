# PostureGuard 배포 전 체크리스트

## HTTPS

- [ ] 배포 URL이 HTTPS로 제공된다.
- [ ] localhost 개발 환경에서 카메라가 동작한다.
- [ ] Vercel 배포 URL에서 카메라 권한 요청이 정상 동작한다.
- [ ] Netlify 배포 URL에서 카메라 권한 요청이 정상 동작한다.
- [ ] GitHub Pages 배포 URL에서 정적 파일 경로가 정상 동작한다.

## 카메라 권한

- [ ] 권한 허용 시 카메라 프리뷰가 보인다.
- [ ] 권한 거부 시 친절한 안내 카드가 보인다.
- [ ] `다시 시도` 버튼으로 권한 재요청 흐름을 확인했다.
- [ ] 카메라가 없는 환경에서 안내 메시지를 확인했다.
- [ ] 다른 앱이 카메라를 사용 중일 때 안내 메시지를 확인했다.

## PWA

- [ ] manifest가 로드된다.
- [ ] 데스크탑 설치 버튼 또는 브라우저 설치 UI가 표시된다.
- [ ] 설치 후 standalone 창으로 실행된다.
- [ ] 아이콘이 정상 표시된다.

## 오프라인

- [ ] 최초 온라인 접속 후 새로고침해 service worker가 등록된다.
- [ ] 네트워크 차단 후 앱 shell이 열린다.
- [ ] 이미 캐시된 리소스는 오프라인에서 로드된다.

## 핵심 기능

- [ ] 기준 자세 저장
- [ ] 모니터링 시작/중지
- [ ] Orb 클릭으로 preview 진입
- [ ] 자세 점수 및 통계 갱신
- [ ] 눈 휴식 알림
- [ ] 스트레칭 알림
- [ ] 경고음 끄기
- [ ] 데이터 초기화

- [ ] Windows Edge
